
# 물류관리 솔루션 v1.0
더존비즈온 JAVA 개발자 채용 교육 과정

개발기간: 2023.08 ~ 2023.10

</br>

  ## 주요 기능
  #### 생산관리
  - 생산품 등록
  - 생산내역 조회
  - 생산라인 관리
  #### 입출고관리
  - 입고 예정
  - 입/출고 확인
  - 입/출고내역 조회
  #### 재고관리
  - 기초재고등록
  - 재고이동
  - 재고조회
  #### 기준정보관리
  - 회사정보/거래처 관리
  - 공통코드 관리
  - 단가관리
</br>
